# cisco-packet-tracer-lab
Physical Layer Implementation using Copper, Fiber and Wireless in Cisco Packet Tracer

### Technologies & Media Used
- **Copper (UTP)**: 100 Mbps & 1 Gbps Ethernet
- **Fiber Optic**: 1 Gbps link between Switch and Router
- **Wireless**: IEEE 802.11ac (Wi-Fi 6) at 64 Mbps
- Modules added to PT-Empty devices
- Proper cable selection (Straight-Through, Cross-Over, Fiber)

### Network Topology
- Laptop connected via Wireless to Access Point
- PC connected via Copper to Switch
- Switch connected to Router via Fiber
- Router connected to Server via Copper

### Key Learnings
- Adding network interface modules
- Configuring wireless security (WPA2-PSK + AES)
- Difference between Straight-Through and Cross-Over cables
- Enabling interfaces
- Packet flow visualization in Simulation mode

**Tools**: Cisco Packet Tracer
